# QR Code NFC Reader Executables

This repository hosts the compiled executables and drivers for the **QR Code & NFC Reader** application.

## Contents

### Executables
- **Windows**: Pre-compiled `.exe` file for Windows platforms
- **macOS**: Pre-compiled application for macOS platforms

### Drivers
- Known working NFC reader drivers for various operating systems
- Driver installation instructions and compatibility information

## About the Application

The QR Code & NFC Reader is a desktop application that allows users to:
- Read QR codes using the device camera
- Read NFC tags using compatible NFC readers
- Process and display the data from both QR codes and NFC tags

## Getting Started

1. Download the appropriate executable for your operating system
2. Install any required NFC reader drivers (if using NFC functionality)
3. Run the application and follow the on-screen instructions

## Driver Installation

### Windows
- Download the appropriate Windows driver
- Run the installer as administrator
- Connect your NFC reader device
- The application should automatically detect the NFC reader

### macOS
- Download the appropriate macOS driver
- Follow the installation instructions provided
- Some drivers may require disabling System Integrity Protection (SIP)

## Troubleshooting

If you encounter issues with NFC functionality:
1. Ensure the correct drivers are installed for your operating system
2. Verify that your NFC reader is compatible with the application
3. Check that the NFC reader is properly connected and recognized by the system
